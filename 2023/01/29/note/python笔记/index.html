python 的学习笔记

黑马程序员python教程-黑马，偶尔的s

pycharm 使用

  • 多行注释快捷键:选中要注释的代码然后按住 shift 键再连按 3 次引号键
  • 保存时自动格式化代码:在设置>工具>保存时的操作,在此处将格式化勾选上即可

数据类型

字面量

  • 被写在代码中固定的值称之为字面量
  • 类似于 js 中的常量

类型查看

  • 使用 type()函数可以查看数据的类型
  • 变量没有类型,但变量存储的数据有类型

类型的转换

  • int():将数据转换为 number 类型
  • str():将数据转换为 string 类型
  • float()

字符串类型(string)

  • 三种定义方式:

  • 字符串的占位符输出: print('我叫%s今年%s岁' % (name, age))

  • 常用占位符: %s,%d,%f

  • 字符串格式化的精度控制:print('11.639格式化后:%5.1f' % 11.639)

    • 格式为:%m.nd,m 用来限制长度当 m 小于数据长度时不生效，n 用来限制小数的长度(四舍五入)

  • python 版的模板字符串:print(f'我叫张三今年{age + 10}岁')

标识符

  • 标识符是在编程过程中所运用的变量名字
  • 标识符的命名规则:英文,数字,中文(不推荐),下划线。(能支持中文是真的骚)
  • 大小写敏感,不能以数字开头

算数运算符

循环

while 循环

  • 和 js 的差不多

for 循环

  • 差点有点多
  • 无法自定义循环条件
  • 相当于 js 中的 forEach 只能用来遍历变量
  • range 函数：
    • 用来生成一个数字序列
    • range(num):生成从 0 开始到 num 的数字序列但不包括 num，如 range(4)=>[0,1,2,3]
    • range(num1,num2):生成一个从 num1 开始到 num2 结束的数字序列
    • range(num1,num2,setup):生成一个从 num1 到 num2 的等差数列，差为set up
  • 作用域:在 for 循环内定义的变量规范上不让外部访问，但实际上是可以访问到的

函数

  • 如何定义函数:使用def关键字来定义函数
  • 函数内的变量属于局部变量
  • 当函数没有 return 时，默认返回 return None,也可以手动返回 return None
  • 函数的传参形式

数据容器

  • 用来存储多个数据的变量

列表(list)

  • 类似于 js 的数组
  • 访问数据时下表可以为负

元组(tuple)

  • 类似于不可二次更改的列表
  • 元组定义:
  • 元组跟列表一样支持嵌套
  • 元组的常用方法:
    • index():返回给定元素的下标
    • count():统计给定元素出现的次数
    • len():返回元组的长度

字符串(string)

  • python 中字符串最骚的一个点就是支持下表查找
  • 常用方法:
    • replace:对字符串进行查找替换
    • split:以给定的字符为界将字符串分割为几个子串，给定的部分会被删除，然后将字串塞入一个列表后返回
    • strip:去除字符串前后当中的给定部分，如果不给参数，则默认去除字符串的前后的空格

序列

  • 定义:内容连续、有序、支持下标索引的一类数据容器
  • 以下容器可视为序列:
    • 列表
    • 元组
    • 字符串
  • 序列切片:
    • 格式:序列[起始:结束:步长]
    • 起始可以省略:省略的话从头开始
    • 结束可以省略:省略的话到末尾才结束
    • 步长默认为 1，当步长为负数时表示倒序取值

集合

  • 集合的定义:
    • 直接一个{}表示定义一个集合
    • 调用函数set()来定义集合
  • 特点：元素不允许重复，内容无序，自带去重功能
  • 集合方法:
    • add():用来添加元素，加在末尾
    • remove():用来移除元素
    • pop():随机从集合中取出一个元素,去除元素后，该元素会从集合中删除
    • clear():顾名思义，用来清空集合
    • difference:用来取差集
    • difference_update:用来去除交集
    • union:用来合并两个集合，返回合并后的集合，不会对原集合进行修改

字典

  • 含义:用来保存key-value的容器，类似于 js 中的 map 对象
  • 如何定义字典:
    • 使用{}来定义 (有点逆天，跟集合重复了,还有当只有一个大括号时默认它是字典类型)
    • 使用函数定义:myDict = dict()
    • key不允许重复
  • 字典的操作(经典增删改查)
    • pop:删除给定 key 的 value 值，并将其 value 值返回
    • clear:用来清空字典
    • keys:stry.keys()不用传入参数，将字典中所有的 key 构成一个列表后返回

容器总结

  • 使用 max()方法可以找到容器中最大的值
  • 同理使用 min()方法可以找到容器中最小的值
  • 容器之间相互转换:
    • list(容器):将容器转换为列表
    • str():转换为字符串
    • tuple():转换为元组
    • set():转换为集合
  • sorted(容器,reverse=False):能对容器进行排序并将排序好的内容构成一个列表后返回
    • 参数reverse默认为 False，可以不写，当为 False 时，将内容从小到大排列,当为 True 时从大到小排列
  • 字符串如何比较大小:
    • 通过 ascll 码来比较大小
    • 字符串是进行按位比较，一旦出现某一位的字符不相等那比较结果就出来了后续的大小不管

文件

文件的编码

  • 编码技术:即翻译的规则，记录了如何将内容转化为二进制，以及如何将二进制转化为可读内容
  • 常见编码技术:
    • UTF-8(常用)
    • GBK
    • Big5

文件的读取

  • 当对一个文件连续调用多次方法来读取后续的读取方法会接着前面的进度来继续读取。

  • read():读取给定数量的字符,如果没有则全部读完,将读取到的内容以字符串返回

  • readline():只读取文件的第一行内容

  • readlines():一行一行的读取全部内容，并将读取到的内容以列表的形式返回

  • close():关闭文件的读取，解除对文件的占用

  • with open语法:好处是能够自动完成关闭读取的操作

文件的写入

  • write方法
  • 注意点
    • 直接调用write，内容是不会真正写入到文件当中，而是会积攒在程序的内存中，称之为缓冲区
    • 当调用flush方法时，内容才会真正的写入到文件中
    • 这样做的原因是:避免频繁写入导致频繁的操作硬盘，导致效率下降
    • close():自带flush功能
    • 该模式下没有文件会创建该文件，且写入是覆盖性写入
  • 追加性操作:
    • 打开文件时注明以追加的方式打开即可:open('demo.txt', 'a', encoding='UTF-8')
    • 该模式下文件不存时，也会自动创建
    • 可以使用\n来写出换行符

异常

  • 异常的具有传递性:基于函数之间的调用方式逐级向上抛出异常

模块

  • 啥叫模块，参考js中的模块定义
  • python如何引入模块:
    • import time:直接引入一整个模块
    • from time import localtime:从一个模块中引入一个类或者方法来使用
    • from time import localtime as mytime:给引入的类或者方法重新起一个名字叫mytime
    • 引入模块时会自动执行模块中的代码，from的时候就已经执行了
