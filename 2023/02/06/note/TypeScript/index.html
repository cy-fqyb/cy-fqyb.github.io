TypeScript 介绍

Ts 是微软开发的一款基于 Js 的静态类型编程语言，是 Js 的超集，特点如下

  • Ts 是在 Js 的基础上进行拓展的，所以 Ts 完全支持 Js 代码
  • Ts 拓展的功能是给 Js 加了类型限制，是 Js 由动态类型转换为了静态类型语言，所以可以说ts = type +JavaScript

Ts 和 Js 的区别

  • Ts 为静态类型语言，而 Js 为动态类型语言，即 Ts 的变量是有类型的而 Js 的变量是没有类型的可以任意存值。
  • 所以 Ts 可以简单的理解为有类型限制的 Js

Ts 开发环境

  • 如何安装 ts

  • 安装成功后输入:tsc ts 文件名 运行后即可将 ts 文件编译为 js 文件

  • 开发软件选择:推荐 VsCode，毕竟也是微软旗下的,天生支持 Ts。

    • VsCode 的配置:安装Cood Runner插件

    • 建议安全局安装ts-node模块，有了它之后就可以直接在 node 环境下运行 Ts 代码而不用编译

    • ```shell
      npm i -g ts-node # 全局安装包
      ts-node ts文件名 # 当想要运行ts代码时，先这样即可，或者可以使用cood runner插件运行

Ts 的类型

类型限制，Ts 的精髓，也是学习 Ts 的最重要的一部分

Js 原有类型:

  • 值类型:字符串（String）、数字(Number)、布尔(Boolean)、空（Null）、未定义（Undefined）、Symbol
  • 引用类型:对象(Object)、数组(Array)、函数(Function)等等。

Ts 新增类型:

  • Any 类型(任意类型):给某一变量声明为 any 类型后，代表这个变量可以存储任意的值，相当于没有了类型限制

    • 显示 Any:即直接声明该变量是 any 类型如let num:any=123
    • 隐式 Any:即创建变量时不声明类型也不赋值如let d这是编译器会判断 d 为隐式 any 类型
    • any 类型的变量会影响别人，它的值可以随意赋值给任意类型的变量，如

  • unknown 类型(未知类型):也可以存储任意类型

    • unknown 类型的变量不能直接赋值给其它类型的变量，可以直接赋值给unknown或者any类型
    • unknown 类型的对象不能调用对象内的属性或者方法，而 any 类型却可以。

  • 接口类型(interface)

    • 可以理解为定义一个规则约束，如果想让变量(一边是对象)使用该类型，必需满足该规则约束。
    • 必需跟接口的属性一模一样，多了不行，少了也不行。

    • 如果存在两个名字一样的接口它会自动将其合并。
    • 任意值:当不确定其它值的类型时可以使用任意 key，
    • 可选值:属性的后面加个问好代表该值可选
    • 只读属性:顾名思义，即该属性只能读取不能修改，在属性名前加上一个 readonly 即可readonly sex: string;
    • 接口继承
    • 接口定义函数类型

  • 对象类型

    • object:用来声明变量后，变量只能接收引用类型的值如对象，数组，函数等等，不能接收值类型变量
    • Object:可以存储任意类型的一个 Object 对象
    • {}:作用和 Object 一样，都是声明一个可以存储任意类型的对象变量

  • 数组类型

    • 普通类型:let arr:类型[]表示定义啥类型的数组
    • 数组泛型:let arr2: Array<number> = [123];表示定义一个数字类型的数组

  • 函数类型声明

    • 函数重载

  • 联合类型:

    • 即声明一个变量可以存储多种类型的值
    • let phone: number | string = "12346"像这样用一个竖线分隔代表或者的意思

  • 交叉类型:

    • 即声明一个变量它保存的值必需满足符合多种指定的类型才行

  • 类型断言(as)

    • 类型断言即当一个变量的类型编译器无法判定但开发者自己清楚时可以使用类型断言

  • 内置对象

    • ECMAScript 的内置对象
    • DOM 和 BOM 的内置对象

  • 元组类型

    • 元组（Tuple）是固定数量的不同类型的元素的组合。
    • 当访问元素时，可以得到元素的数据类型
    • 当添加越界元素时，会限制添加的元素只能是指定的类型

  • 枚举(enum)

    • 是一种特殊类
    • 枚举（Enum）类型用于取值被限定在一定范围内的场景
    • 用来给存储的数据取一别名，用来减少数据库的占用

  • 类型别名

    • 顾名思义就是给类型另起一个名字便于记忆或其它

泛型

泛型 · TypeScript

  • 概念:是指在定义函数、接口或类的时候，不预先指定具体的类型，而在使用的时候再指定类型的一种特性。

  • 泛型约束:

    • 在函数内部使用泛型变量的时候，由于事先不知道它是哪种类型，所以不能随意的操作它的属性或方法
    • 通过泛型约束来指定传入的类型变量中必需包含某属性或方法才行

  • 使用keyof约束对象

    • 说人话就是用对象的键作为变量的约束类型，是一个联合类型
