go 的笔记

go 的基础学习笔记

变量

  • 全局变量

    在 main 函数外定义的变量就是全局变量

  • 局部变量

    在函数内部定义的变量就是局部变量，局部变量如果声明后不引用则会报错,而全局变量不会

go 当中声明变量的特殊地方

  1. go 声明一个变量时如果不给赋值，则必需声明变量的类型，如:var name string
  2. 如果声明后不赋值，那这个变量会有默认值，go 会自动给其添加默认值
  3. go 具有类型推断，如果声明变量时给它赋了值，则可以不声明类型，因为 go 会自动推断它的类型如var sex = "男"
  4. 变量的简写定义:当定义一个变量且给变量赋值时，可以简写为 sex:="男"
  5. 当声明多个变量时可以使用批量声明

数组

是同一种类型的固定长度的序列

go 当中数组的特殊性

  • go 语言的数组与以往的数组有很大的不同，最明显的一点就是，go 当中的数组是值类型的
  • 长度是数组类型的一部分因此，var a[5] int 和 var a[10]int 是不同的类型。
  • 数组是值类型，赋值和传参会复制整个数组，而不是指针。因此改变副本的值，不会改变本身的值。
